# 软件推荐

> 开发中常用的优秀软件和工具，持续更新。

## 编辑器 & IDE

### Visual Studio Code

- **平台：** Windows / macOS / Linux
- **官网：** <https://code.visualstudio.com/>
- 微软出品的轻量级代码编辑器，插件生态丰富，支持几乎所有编程语言。前端、全栈开发的首选编辑器。

## 数据库

### DBeaver

- **平台：** Windows / macOS / Linux
- **官网：** <https://dbeaver.io/>
- 免费开源的通用数据库管理工具，支持 MySQL、PostgreSQL、Oracle、SQLite 等。社区版已满足日常开发需求。

## 终端 & SSH

### WindTerm

- **平台：** Windows / macOS / Linux
- **官网：** <https://github.com/kingToolbox/WindTerm>
- 开源跨平台的 SSH 终端工具，支持分屏、自动补全、文件传输，性能优秀，界面现代化。
