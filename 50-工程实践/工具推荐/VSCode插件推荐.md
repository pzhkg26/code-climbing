# VSCode 插件推荐

> 提升开发效率的 VSCode 插件，持续更新。

## 主题 & 外观

### Dracula Theme Official

- 经典暗色主题，护眼配色，在 VSCode、JetBrains、iTerm2、Obsidian 等众多软件中均可使用，跨平台体验一致。

### vscode-icons

- 为文件和文件夹提供直观的图标，通过图标快速识别文件类型。

### indent-rainbow

- 给代码缩进层级着色，便于阅读嵌套较深的代码结构。

### Rainbow Brackets

- 给配对括号赋予不同颜色，快速定位括号作用域，避免漏括号。

## 代码质量 & 规范

### Alibaba Java Coding Guidelines

- 阿里巴巴 Java 开发规约插件，实时检测代码是否符合规范，一键修复。

### Error Lens

- 将错误、警告提示内联显示在代码行尾，无需悬停鼠标即可看到诊断信息。

## 工具

### Office Viewer

- 在 VSCode 内直接预览 Office 文档（Word、Excel、PPT），无需切换应用。
